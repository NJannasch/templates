# SLS CodeBuild Example for serverless framework with components

setup npm and install jest
```
npm init -y
npm i jest --save-dev
```

update package.json
```
"scripts": {
    "test": "jest"
}
```

make test folder

```
mkdir __tests__
```

make test file

```
$ touch __tests__/index.spec.js
```

test

```
npm run test
```
